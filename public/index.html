Get started

    API Endpoint

        https://we-now.herokuapp.com/

The We Now API provides programmatic access to the weather data provided by OpenWeather API.

To use this API, you don't need an API key.

Get Data


# Here is a curl example
curl --header "Content-Type: application/json" \
  --request POST \
  --data '{"city":"newyork"}' \
  https://we-now.herokuapp.com/city/query

Get City Data


To get city data you need to make a POST call to the following url :
https://we-now.herokuapp.com/city/query


JSON PARAMETRS


Field Type Description
name String matching the city name

Get City by Location


To get a city nearby a specific coordinates you need to make a POST call to the following url :
https://we-now.herokuapp.com/city/coord


JSON PARAMETRS


Field Type Description
lat String Location Latitude Coordinates
lon String Location Longitude Coordinates

Get Weather by City Id


To get the current weather in a city you need to make a POST call to the following url :
https://we-now.herokuapp.com/weather/id


JSON PARAMETRS


Field Type Description
id String location Id

Get full Weather by Location


To get the full weather data in a city near you, you need to make a POST call to the following url :
https://we-now.herokuapp.com/weather/coord/full


JSON PARAMETRS


Field Type Description
lat String Location Latitude Coordinates
lon String Location Longitude Coordinates
